## Deployment

### Prerequisites
- AWS account with Bedrock access in us-east-1
- Titan Text Embeddings V2 and Claude 3 Haiku available in your account

### Steps

**1. Create S3 docs bucket**
- Create a private general purpose S3 bucket
- Upload your product documentation (`.txt`, `.pdf`, `.md`, `.html`, `.docx`)

**2. Create S3 vector bucket**
- In S3 console â†’ Vector buckets â†’ Create vector bucket
- Inside it, create a vector index with 1024 dimensions and Cosine distance metric

**3. Create Bedrock KB service IAM role**
- Trust policy: `bedrock.amazonaws.com`
- Permissions: `s3:GetObject` + `s3:ListBucket` on docs bucket, `bedrock:InvokeModel` on Titan model ARN, `s3vectors:*` on your vector index ARN

**4. Create Bedrock Knowledge Base**
- Bedrock console â†’ Knowledge Bases â†’ Create â†’ Knowledge Base with vector store
- Data source: your docs S3 bucket
- Vector store: S3 Vectors â†’ select your vector bucket and index
- Embedding model: Titan Text Embeddings V2
- Sync the data source after creation

**5. Create Lambda execution IAM role**
- Trust policy: `lambda.amazonaws.com`
- Permissions: CloudWatch Logs, `bedrock:RetrieveAndGenerate` + `bedrock:Retrieve` on your KB ARN, `bedrock:InvokeModel` on Claude 3 Haiku ARN

**6. Create Lambda function**
- Runtime: Python 3.12
- Attach the execution role
- Paste the handler code from `lambda/handler.py`
- Set timeout to 30 seconds
- Set environment variables:
  - `KNOWLEDGE_BASE_ID` â€” your KB ID from Bedrock console
  - `MODEL_ARN` â€” `arn:aws:bedrock:us-east-1::foundation-model/anthropic.claude-3-haiku-20240307-v1:0`
  - `FRONTEND_ORIGIN` â€” your S3 static site URL (or `*` for testing)

**7. Create API Gateway**
- HTTP API with one route: `POST /chat` â†’ Lambda integration
- Enable CORS: allow your S3 frontend origin, `Content-Type` header, `POST` + `OPTIONS` methods
- Add resource-based permission on Lambda for API Gateway

**8. Deploy frontend**
- Create a public S3 bucket with static website hosting enabled
- Add bucket policy allowing `s3:GetObject` on `/*`
- Update `API_URL` in `index.html` to your API Gateway invoke URL
- Upload `index.html`

## Cost

Designed to be low cost for personal/demo use. At small scale (single doc, low query volume) estimated cost is under $1/month. Main cost drivers are Bedrock model invocations (per token) and S3 Vectors query processing.
